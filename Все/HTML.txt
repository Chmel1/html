HTML-теги

	Загаловки и текст:
		<h1>-<h6> - Загаловки разного уровня. h1 - самый важный h6 - наименее важный
		<p> - определяет абзац текста 
		<span> - используется для стилизации отдельных частей текста 
		br - перенос строки
		hr - горизонтальная линия для разделения контента
		
	Семантические теги:
		header - заголовочная часть страницы или секции
		nav - блок для наивгации
		main - основной контент страницы
		section - логический раздел контента
		article - независимый блок контента 
		aside - соержимое связанное с основным контентомm например, боковая панель 
		footer - нижняя часть страницы 
	
	Работа с медиа:
		img - Отображение изображения
		video - Вставка видео. Поддерживает атрибуты controls, autoplay, loop
		audio - Вставка аудио. Атрибуты как у video
		figure - Блок для изображенй и других медиа с описанием
		figcaption - Подпись к figure
		
	Формы:
		form - Контейнер для формы
		input - Поля ввода
		label - Ассоциация подписи к элементам формы 
		textarea - Многострочное текстовое поле
		button - Кнопка
		select - Выпадающий список 
		option - Элемент выпадющего списка 
		
	Ссфлки и списки:
		a - Гиперссылка
		ul - Ненумерованный список
		ol - Нумерованный список 
		li - Элемент списка 
		
	Таблицы:
		table - Создание таблицы
		tr - Определяет стркоу таблицы 
		td - Ячейка данных 
		th - Загаловочная ячейка 
		caption - Заголовок таблицы 
		
	Полезные теги:
		div - Контейнер для группирования элементов. Обычно используется для стилизации и разметки.
		iframes - Встраивание другого HTML-документа на страницу
